Random Color Palette Generator - Adobe Illustrator Extension
The Random Color Palette Generator is an Adobe Illustrator extension that allows users to effortlessly generate random color palettes and design styles for their creative projects. This documentation provides detailed instructions on how to install and use the extension effectively.

Installation
To install the Random Color Palette Generator extension in Adobe Illustrator, follow these steps:

Clone or download the repository to your local machine.
Open Adobe Illustrator.
In the menu bar, go to Window > Extensions.
Click on the Browse button and navigate to the location where you saved the extension files.
Select the extension file (RandomColorPaletteGenerator.aip or similar) and click Open.
The extension will be installed and appear in the Extensions panel within Adobe Illustrator.
Usage
To use the Random Color Palette Generator extension, follow these steps:

Open Adobe Illustrator.
In the menu bar, go to Window > Extensions > Random Color Palette Generator.
The extension interface will open, displaying a random color palette and design style.
Customization
The Random Color Palette Generator extension allows you to customize the generated themes according to your preferences. Follow these instructions to customize the color palette and design style:

To generate a new random color palette, click the Generate Color button.
To generate a new random design style, click the Generate Design button.
You can further modify the color palette and design style as desired using the following options:

Color Palette: Each color in the palette is represented as a circle. You can click on a color circle to select it and use the color picker to choose a new color.
Design Style: The available design styles include vintage, futuristic, minimalistic, abstract, modern, elegant, hand-drawn, geometric, grunge, typography-based, watercolor, and more. Click on the desired design style to apply it to the current theme.
Saved Themes
The Random Color Palette Generator extension allows you to save and manage your customized themes for future use. Follow these instructions to save and utilize the saved themes:

To save the current theme, enter a name in the "Theme Name" input field and click the Save Theme button.
Saved themes will appear in the "Saved Themes" section below, displaying the theme name, color palette, design style, and a "Delete Theme" button to remove a saved theme.
You can search for specific themes by entering a search term in the "Search" input field. The filtered themes will be displayed below.

Tech Stack
The Random Color Palette Generator extension is built using the following technologies:

React: A JavaScript library for building user interfaces.
TypeScript: A statically-typed superset of JavaScript that compiles to plain JavaScript, providing enhanced developer tooling and improved code maintainability.
Code Structure
The code for the Random Color Palette Generator extension is structured as follows:

App.tsx: The main component that defines the user interface and handles the extension's functionality.
App.css: The CSS styles associated with the extension's user interface.
index.tsx: The entry point of the extension, responsible for rendering the App component.
